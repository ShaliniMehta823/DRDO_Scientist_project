# DRDO Scientist Project

##  Overview

This project is a Machine Learning-based implementation developed as part of my learning and practical application of data science concepts. The notebook demonstrates the complete ML workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model building, evaluation, and prediction.

The primary objective of this project is to apply machine learning techniques to solve a real-world problem using Python and popular data science libraries.

---

- Data Loading and Exploration
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)

- Well-documented Jupyter Notebook implementation

## How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/ShaliniMehta823/DRDO_Scientist_project.git
```

### 2. Navigate to the Project Folder

```bash
cd DRDO_Scientist_project
```

### 3. Install Dependencies

```bash
pip install numpy pandas matplotlib seaborn scikit-learn jupyter
```

### 4. Launch Jupyter Notebook

```bash
jupyter notebook
```

Open:

```
DRDO_Scientist_project.ipynb
```
